# Roadmap

## Completed

### SPF-001 — Basic MVP

- File and manual-text input
- Cleaning, bilingual pain detection, emotion scoring
- Streamlit dashboard, sample data, and basic tests

### SPF-001-CN — Chinese-first UI

- Chinese-first labels and bilingual category display
- Improved compliance messaging and readability

### SPF-002 — Business opportunity scoring

- Evidence-based category scoring
- Top 5 opportunities, business explanations, charts, and enhanced ideas

### SPF-002B — Business interpretation polish

- Market-signal strength and urgency
- Monetization, scenarios, portfolio value, validation suggestions
- Founder / Product Lens

### SPF-003 — Report export and local outputs

- Timestamped CSV, Markdown, and standalone HTML exports
- Explicit local saving to `outputs/`
- Excel-compatible UTF-8-sig CSV

### SPF-004 — Product-grade UI polish

- Product Hero and structured sidebar
- Six core metrics and grouped charts
- Screenshot-ready opportunity cards and numbered dashboard flow

### SPF-005 — GitHub public showcase

- Portfolio-ready README
- Architecture, privacy, roadmap, changelog, and screenshot documentation
- Public showcase manifest and release-safety checks

## Potential future versions

- Expand and customize multilingual keyword dictionaries
- Add user-editable categories and scoring weights
- Add segment, source, and time filters
- Add more report layouts and comparison views
- Add optional OpenAI / LLM interpretation behind an explicit opt-in
- Add PDF export after establishing stable licensed Chinese-font handling
- Add an optional local database for private installations
- Add a synthetic-data-only deployment demo
- Add human review and correction workflows
- Add local semantic models where hardware allows

Future work must preserve consent, explainability, privacy, and clear separation between research signals and verified market demand.
